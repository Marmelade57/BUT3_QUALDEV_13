Pour lancer le test Selenium présent en Java dans ce projet, faites un clic droit sur le fichier et executez le avec 'Run SeleniumLoginTest'. 
Sinon, suivez la procédure suivante :

- installez Selenium IDE sur votre navigateur
- lancez l'extension
- sélectionnez l'option pour ouvrir un projet existant
- recherchez le fichier .side correspondant au projet
- lancez les tests via le bouton play
